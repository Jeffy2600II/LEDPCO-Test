LINE Bot — หน้าแสดงสถานะ (พื้นฐาน)

หน้านี้เป็น HTML แบบปกติ (ไม่มี JavaScript) สำหรับแสดงคำแนะนำและสถานะแบบเบื้องต้น

วิธีตรวจสอบการเชื่อมต่อ

  1. ตั้งค่า Webhook URL ใน LINE Developer Console ให้ชี้ไปที่: 
    https://YOUR_PROJECT.vercel.app/api/webhook
  2. ส่งข้อความมาที่ bot แล้วตรวจดู Reply ใน LINE หรือดู Logs ใน Vercel (ถ้า deploy)
  3. ถ้ารัน local ใช้ ngrok: e.g. ngrok http 3000 แล้วตั้ง webhook เป็น https://<ngrok-id>.ngrok.io/webhook

คำสั่งพื้นฐานที่รองรับ

  • help / ช่วย
  • ping → pong
  • สวัสดี → ทักทาย
  • echo <ข้อความ> → ส่งข้อความกลับ
  • การคำนวณ เช่น 2 + 3 หรือ 2 บวก 3
  • status / สถานะ → แสดงเวลาที่ webhook ถูกเรียกล่าสุด

หมายเหตุ: สถานะเก็บแบบไฟล์ชั่วคราว (data/status.json หรือ /tmp/status.json) — ถ้าต้องการเก็บถาวร ให้เชื่อมฐานข้อมูลจริง

ถ้ามีปัญหา "Not Found" สำหรับเส้นทาง / ให้ตรวจสอบว่าไฟล์ index.html อยู่ที่ root ของ repo หรือมี rewrite ให้ไปที่ /api/index.js
